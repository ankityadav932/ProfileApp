Javascript

Javascript is a programming language for the browser. From accessing the device's camera, GPS, etc. to making HTTP requests and creating socket connections, it has all the APIs built-in that the browser can provide. I have worked a lot on Javascript. Most of my work on Javascript contains DOM manipulation and making HTTP requests to the server.

PHP

PHP is the programming language of the server. It provides a great and easy way to code your application's backend. What I really love about PHP is its ability to fit itself inside tags, which means it can adjust itself to other languages inside a single file. Combined with MySQL, it makes a deadly combination as it has a lot of built-in functions for accessing the MySQL database, and that is the very same combination I have worked on.

Laravel

Laravel is a full-stack framework based on PHP. What I have experienced about Laravel is that it's the best framework I have ever seen. From using Blade as a front-end templating engine to having features like Eloquent ORM, authentication, and Artisan (a built-in command-line tool), it is a one-stop solution for any requirement you have in mind. I had done little work on it as a professional, but I had read this framework in good detail.

NodeJS & Express

NodeJS is a runtime environment that provides the ability to use JavaScript as a server programming language, and Express is a framework that sits on top of NodeJS to provide additional features like routing, middleware, etc. NodeJS is event-driven and asyncronous, which makes it work smarter than other backend technologies. I am quite new to NodeJS, and I am learning this amazing technology.

React

React is a front-end Javascript that allows you to build single-page applications divided into multiple reusable and flexible Javascript components. It provides a real-time essence to the application and an optimized way to work with the application frontend. I am quite new to React, and I am learning this amazing technology.

CSS (UI)

CSS is a style sheet language that is used to control the layout and design of an HTML document. I have been working on CSS since the very beginning of my coding journey and have a good command of it. I had also worked on two very popular CSS frameworks, Bootstrap and Tailwind CSS. I had worked on some complex UI design requirements and understood how things work in CSS.

MySQL

MySql is an RDBMS (relational database management system) that works with SQL (structured query language). MySql, or any SQL database, is used to store data units that are interrelated with each other. I have worked a lot with MySQL databases using basic concepts because that's enough to cater to most applications requirements. As programming is my primary focus, I didn't get into advanced SQL concepts.

MongoDB

MongoDB is a document-oriented DBMS (database management system) that stores each data unit as a separate document in the form of BSON (binary-encoded JSON) format. It is used to store data units that are not interrelated. It is faster than relational databases as it doesn't have to maintain any relationships between data units. I am quite new to MongoDB, and I am learning this amazing technology.

Git & Github

Git is a program that provides the ability to version control your software development process, and Github is a platform that provides an online space to store the entire codebase with all the versions that have been developed and deployed after Git initialization in your project. I have worked with Git, and have good knowledge of this skill.
